Mini-cours : Bases du HTML et CSS

Cette page présente quelques balises HTML de base, avec explications simples pour débutants.

Les balises de titre <h1> à <h5>

<h1>Titre principal</h1>
<h2>Sous-titre</h2>
<h3>Titre de section</h3>
<h4>Titre plus petit</h4>
<h5>Titre encore plus petit</h5>
- <h1> : Le titre le plus important d'une page. - <h2> : Sous-titre, généralement utilisé pour les grandes sections. - <h3> : Pour les sous-sections d’un <h2>. - <h4> : Encore plus petit, pour détailler davantage. - <h5> : Un titre encore moins important, rarement utilisé.

⚠️ Il est conseillé d’utiliser les titres dans l’ordre hiérarchique (h1, puis h2, etc.), pour que la page soit bien structurée.

Le paragraphe avec un lien hypertexte

<p>Un paragraphe contenant un lien vers <a href="https://www.webacademy.com">WebAcademy</a>.</p>
- <p> : Balise d’ouverture d’un paragraphe. - </p> : Balise de fermeture du paragraphe. - <a> : Balise de lien (ancre). - href : Attribut qui indique l’adresse du lien. - Contenu du paragraphe : Le texte affiché à l’écran.
