Introduction au HTML

Les éléments essentiels d'une page HTML

<!DOCTYPE html> : indique au navigateur la version HTML.

Comme un panneau indiquant "HTML5"

<html lang="fr"> ... </html> : englobe tout le contenu et indique la langue.

Racine de la page

<head> ... </head> : contient les informations pour le navigateur (non visibles).

Informations techniques

<meta charset="UTF-8"> : définit l'encodage des caractères.

Caractères affichés correctement

<meta name="viewport" content="width=device-width, initial-scale=1.0"> : rend la page adaptée aux mobiles.

Page adaptée à tous les écrans

<title> ... </title> : définit le titre visible dans l’onglet du navigateur.

Nom de la page dans l'onglet

<body> ... </body> : contient tout ce que l’utilisateur verra (texte, images, vidéos).

Contenu visible de la page

Exemple d’un paragraphe avec un lien

<p>Un paragraphe avec un lien vers 
<a href="https://www.webacademy.edu">WebAcademy</a>.</p>

<p> → balise d’ouverture du paragraphe

</p> → balise de fermeture du paragraphe

href → attribut indiquant l’adresse du lien

Le texte du paragraphe → contenu visible

L’ensemble → un paragraphe HTML contenant un lien cliquable

Cliquer sur le lien ouvre la page

Résumé

Ces éléments sont la base de toute page HTML. Toujours les inclure pour que votre page fonctionne correctement sur tous les navigateurs et appareils.
