Semaine 01 : Activité - Travailler avec les images sur le Web

Présentation

La plupart des sites web contiennent non seulement du texte, mais aussi des images. Certains formats d’images sont plus adaptés au web que d’autres. Cette activité introduit l’utilisation d’images pour le web et explique comment les optimiser afin qu’elles soient plus légères et que les pages se chargent plus rapidement.

Préparer

Types d’images

Les formats d’images les plus courants sur le web sont :

  • JPEG / JPG : idéal pour les photos et images complexes. Compression avec perte, mais taille réduite.
  • PNG : utilisé pour les graphiques, logos et images avec fond transparent. Compression sans perte.
  • GIF : pour de petites animations et graphiques simples. Limité à 256 couleurs.
  • WEBP : format moderne, compression efficace, supporte transparence et animation.
  • SVG : parfait pour les logos, icônes et textes qui doivent rester nets en toutes tailles.

Inclure des images dans une page

Exemples d’utilisation de la balise <img> :

<img src="temple.jpg" alt="Photo du temple">
<img src="processus.png" alt="Schéma du processus de l’entreprise">
<img src="logo.svg" alt="Logo de l’entreprise">

L’attribut alt est essentiel : il décrit l’image si elle ne s’affiche pas, aide les moteurs de recherche et améliore l’accessibilité pour les personnes utilisant des lecteurs d’écran.

Optimisation des images

Les images trop lourdes ralentissent le chargement d’une page. Il est donc important de réduire leur taille sans perdre en qualité visuelle.

Trois étapes de base pour optimiser une image

  1. Recadrer l’image pour ne garder que l’essentiel.
  2. Redimensionner à la taille nécessaire pour la mise en page.
  3. Réduire la qualité jusqu’à obtenir une taille de fichier acceptable.

Instructions de l’activité

Dans cette activité, vous allez optimiser une image trop lourde puis l’intégrer dans une page web.

Étapes :

  1. Ouvrir votre dossier de projet dans VS Code.
  2. Importer une image trop grande dans le dossier images.
  3. Utiliser un outil gratuit en ligne comme Squoosh.app pour la compresser.
  4. Enregistrer l’image optimisée dans le même dossier.
  5. Mettre à jour le code HTML avec la nouvelle version plus légère.

À retenir

Optimiser ses images est une bonne pratique indispensable : cela rend vos pages plus rapides, plus accessibles et agréables à utiliser.
