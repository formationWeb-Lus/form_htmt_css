S01 Activité d’apprentissage : Introduction au HTML

Aperçu

Chaque fois que vous visitez une page web dans votre navigateur, votre ordinateur envoie une requête à un serveur sur Internet. Le serveur répond en renvoyant un fichier que votre ordinateur interprète et affiche dans le navigateur. Le fichier le plus courant est le HTML, qui permet de structurer l’information afin qu’elle soit correctement affichée.

HTML signifie HyperText Markup Language (langage de balisage hypertexte) et fait partie des trois technologies de base du Web : HTML, CSS et JavaScript. Dans ce cours, vous apprendrez les fondamentaux du HTML et du CSS afin de créer des sites web efficaces et professionnels.

Le HTML définit la structure et le sens d’un document web. L’hypertexte fait référence aux liens hypertextes qui permettent de naviguer d’une page à l’autre. Le balisage est un ensemble de symboles ou de codes permettant d’afficher du contenu sur Internet. Les navigateurs web, comme Google Chrome, utilisent le HTML pour rendre les pages visibles.

Concepts essentiels du HTML

Voici un exemple simple d’une page HTML :


<!DOCTYPE html>
<html lang="fr">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mes choses préférées</title>
  </head>
  <body>
    <h1>Mes choses préférées</h1>
    <p>Voici une liste de certaines de mes choses préférées.</p>
    <p>J'aime ma famille et le temple. Voici quelques photos :</p>

    <img src="images/family.jpg" alt="Ma famille">
    <img src="images/salt-lake-temple.jpg" alt="Le temple de Salt Lake">
  </body>
</html>

Comme vous pouvez le voir, une page web se compose d’un élément html contenant deux sections principales : head et body. L’élément head contient des informations importantes pour le navigateur, tandis que body contient tout le contenu visible par l’utilisateur.

Section Head

  • <meta charset="utf-8"> : définit l’encodage des caractères.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0"> : permet un rendu correct sur tous les appareils.
  • <title>...</title> : définit le titre affiché dans l’onglet du navigateur.

Section Body

  • <h1> ... </h1> : titre principal de la page.
  • <p> ... </p> : paragraphe de texte.
  • <img src="..." alt="..."> : insertion d’une image avec un texte alternatif.

Exercice pratique

Essayez d’ajouter :

  • Votre couleur préférée dans un paragraphe.
  • Un nouveau paragraphe avec le texte : J’apprends le développement Web !
