Semaine 01 - Activité d’apprentissage : Travailler avec les images Web

Aperçu

La plupart des sites Web contiennent non seulement du texte, mais aussi des images. Certains formats d’images sont mieux adaptés au web que d’autres. Cette activité introduit l’utilisation des images pour le web et explique comment les optimiser afin qu’elles utilisent moins de mémoire, réduisant ainsi le poids global de la page et le temps de téléchargement.

Préparation

Types d’images

La vidéo suivante présente les types d’images les plus utilisés sur le web :

Comme expliqué dans la vidéo, les types d’images les plus courants pour le web sont :

  • JPEG / JPG : Utilisé pour les photos et images complexes. Compression avec perte pour réduire fortement la taille du fichier.
  • PNG : Utilisé pour les graphiques, logos et images nécessitant un fond transparent. Compression sans perte et support de la transparence alpha.
  • GIF : Utilisé pour les petites animations et graphiques simples. Limité à 256 couleurs, mais prend en charge la transparence de base et les animations.
  • WEBP : Format moderne offrant une meilleure compression que JPG et PNG, tout en supportant animation et transparence.
  • SVG : Utilisé pour les logos, icônes et textes nécessitant un redimensionnement sans perte de qualité.

Inclure des images dans une page web

Pour inclure une image sur une page web, on utilise la balise <img> :

<img src="temple.jpg" alt="Photo du temple">
<img src="notre-processus.png" alt="Schéma décrivant le processus de l’entreprise">
<img src="logo.svg" alt="Logo de l’entreprise">

L’attribut alt est très important car il décrit l’image si elle ne s’affiche pas, aide au référencement et est utilisé par les lecteurs d’écran.

Optimisation des images

Il est essentiel d’optimiser les images pour améliorer l’expérience utilisateur. Des images trop grandes ralentissent le chargement et consomment plus de données.

Trois étapes de base de l’optimisation

  1. Recadrer l’image pour garder la partie importante.
  2. Redimensionner l’image à la taille maximale réellement nécessaire.
  3. Réduire la qualité à un niveau acceptable pour alléger le fichier.

Optimiser ses images est un moyen simple et rapide d’améliorer les performances d’une page.

Dans ce cours, toutes les images doivent être optimisées et faire moins de 100kB.

Instructions de l’activité

  1. Trouvez le fichier temple-large.jpeg dans le dossier images.
  2. Ouvrez Squoosh.app.
  3. Redimensionnez l’image à 800 pixels de large.
  4. Réduisez sa taille jusqu’à moins de 100kB.
  5. Enregistrez-la sous temple-small dans le dossier images.

Mettre à jour le fichier HTML

  1. Ouvrez le fichier images.html.
  2. Modifiez la balise <img> pour afficher votre nouvelle image.
  3. Rechargez la page et comparez avec l’image originale.
